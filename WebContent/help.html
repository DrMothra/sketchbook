Sketcher Help

Overview

A Sketch comprises a set of Objects.

An Object has a title and description (the title is the first line of the description), and a drawing.

An Object drawing can include (references to) other objects. For example one Object might represent a user, another a device, a third Object might then show the user and the device in a certain context.

An Object drawing can include named Frames. These are rectangular regions which are highlighted as areas of particular interest, analogous to the frames in a cartoon.

Frames can be organised into Sequences (not yet implemented), which can represent possible sequences of events or narrative flow.

There can in principle be many Sequences.

Drawings can also be annotated (not yet implemented) for various purposes, for example to raise questions about what is shown or identify key issues

In principle (not yet implemented) Sketches could be developed collaboratively, and track changes made and who has made those changes, for example in a collaborative design process.

User interface

Note that the user interface is non-standard, representing in part some explorations of interaction proposals in Harold Thimbleby's book on User Interface Design

The Index tab shows all current Objects. Objects also appear as tabs if/when they are edited.

The Sequences tab is intended to show and edit sequences of Frames, but is not yet implemented.

In a drawing area the main commands are: (key-press only, no mouse press required)

  • E - edit item
  • A - draw a freehand line item
  • S - insert current Orphan text as text item
  • D - 'place' current selection: if selection is an Object then insert a link to that object, else insert a copy. Hold and drag to specific the size.
  • F - draw a Frame and label with current Orphan text. Hold and drag to specify size.
  • Z - zoom in
  • X - zoom out
  • C - centre/zoom to show all content
  • [Delete] - delete item(s)
  • [Space] - move item(s) to back of drawing stack
  • [LeftMouseButton] - select item(s)
  • [Escape] - move text input focus to Orphan text, i.e. allow text to be typed for following Text or Frame command

Not all commands will work in all areas, but most should (e.g. cannot insert lines or text directly in the Index area).

The Selection history shows recently selected things (items, Objects, images). Selecting within the Selection history will bring that item to the front, e.g. ready to Place.

An image can be inserted by Loading it into the Selection history and then Placing it in an Object

Sketches can be saved as a JSON-encoded data file using the Save and Choose file buttons at the top.
